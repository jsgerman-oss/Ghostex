# zmux 2.1.0

- The 2.x release line continues the full UI refresh around the simplified Codex-style workspace, including refreshed project groups, action controls, tooltips, session cards, settings surfaces, and updated screenshots.
- Stability and performance remain a major focus across native sidebar sync, AppKit relayout avoidance, shared storage writes, diagnostic filtering, and workspace visibility.
- The macOS app now ships with the branded app icon from agent-manager-x in Finder, Dock, app switcher, and signed release bundles.
- The icon is compiled through Xcode's AppIcon asset catalog so local builds and notarized releases use the same icon metadata.
