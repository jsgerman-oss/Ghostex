# zmux 2.0.0

- The whole UI has been refreshed around the simplified Codex-style workspace, including top chrome, project groups, action controls, tooltips, session cards, Previous Sessions rows, settings surfaces, icons, and README screenshots.
- Stability and performance have been improved across native sidebar sync, AppKit relayouts, shared storage writes, diagnostic logging, and workspace visibility.
- Native restore and fork actions are improved, including Codex and Claude fork command support.
- First-prompt auto-renaming now preserves meaningful terminal-synced titles instead of sending redundant generated rename commands.
