## zmux 2.2.0

This minor release tightens the new 2.x interface with the latest workspace, settings, and release workflow polish.

- Unified Settings, Agents, Actions, and Hotkeys into one tabbed settings dialog.
- Added lazy ~/.zmux folder statistics plus an open-folder action in settings.
- Added menu bar workspace status indicators, with optional floating indicators still available.
- Updated session language to use "working" status consistently across the app.
- Improved editor rows with opening and error diagnostics plus clearer retry visibility.
- Polished session cards, close affordances, and Previous Sessions row layout.
- Split developer startup so normal `bun s` is cleaner while `start:dev` keeps the dev app workflow.
- Refreshed README feature coverage for the current 2.2 UI and workspace behavior.
