# Ghostex 3.0.0

- Added a built-in Tips & Tricks guide for workspace basics, agents, actions, Codex setup, and remote access.
- Added bundled `ghostex` and `gtx` command-line launchers so Homebrew installs both commands automatically.
- Changed new session labels to shorter `g-MMDD-HHMMSS` identities that are easier to read in Ghostex, tmux, zmx, and zellij.
- Improved sleeping-session restore behavior so waking a session keeps the current split layout intact.
- Polished project workspace controls, titlebar modes, empty project startup, and project panel behavior.
- Improved native pane and browser stability around focus, tab scrolling, titlebar actions, and embedded browser resizing.
- Improved rename cleanup for pasted session titles.
