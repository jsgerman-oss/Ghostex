## zmux 2.3.2

This patch release adds session attention notifications and tightens project editor row alignment.

- Added optional macOS attention banners when sessions need attention.
- Added a Settings toggle for macOS attention notifications.
- Added native notification permission handling, rate limiting, and click-to-focus routing back into the correct zmux session.
- Kept attention banners separate from completion sounds so clickable system routing can be enabled without audible alerts.
- Improved reference sidebar project editor diff-row alignment.
- Expanded Storybook fixture coverage for open project editor rows with diff stats.
